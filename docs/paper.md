

```bash
Usage: pyzettel paper [OPTIONS]

  Create a zettel from a paper DOI

Options:
  -d, --doi TEXT                  DOI of paper to retrieve  [required]
  -g, --tag TEXT                  Tags for the zettel. Will be added to the
                                  auto generated tags.
  -r, --generate-references / -no-generate-references, -R
                                  Generate references for the publication
  -c, --generate-citations / -no-generate-citations, -C
                                  Generate citations for the publication
  -b, --generate-bibtex / -no-generate-bibtex, -B
                                  Generate bibtex for the publication
  -a, --add-to-bibtex / -A, --no-add-to-bibtex
  -w, --download / -W, --no-download
  --help                          Show this message and exit.
```
